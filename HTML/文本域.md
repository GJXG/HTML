1<textarea> 即文本域
与[文本框](https://how2j.cn/k/html/html-text/191.html)不同的是，文本域可以有多行，并且可以有滚动条

```
<textarea>abc
def
</textarea>
```

# 设置宽度和行数

使用属性cols和rows

```html
<textarea cols="30" rows="8">123456789012345678901234567890
1234567890
1234567890
1234567890
1234567890
1234567890
1234567890
1234567890</textarea>
```
