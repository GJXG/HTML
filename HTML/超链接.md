# 超链

<a>标签即用来显示

超链完整元素是

<a href="跳转到的页面地址">       （超链显示文本）</a>

```html
<a href="https://www.12306.cn/index/">12306</a>
<!--跳转到12306网站-->
```

# 在新的页面打开超链

新增属性target

```html
<a href="https://www.12306.cn/index/" target="_blank">12306</a>
<!--跳转到12306网站-->
```

# 超链上的提示文字

```html
<a href="https://www.12306.cn/index/" title="将跳转到12306">12306</a>
<!--鼠标放在连接上会出现文字-->
```

当鼠标放在超链上的时候，就会弹出提示文字
